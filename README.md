## 🚀 Nova AI — Idea to Prototype Generator

Nova AI is an AI-powered platform that helps transform simple startup ideas into structured product concepts. Users can describe their idea and instantly receive outputs like a startup name, product description, key features, and a pitch deck outline.

The goal of this project is to help founders and innovators quickly move from idea to prototype using artificial intelligence.

## ⚙️ How It Works

User opens the application.

User creates an account or logs in.

User describes their startup idea in the chat.

The AI processes the input using the Groq API.

The system generates a structured startup concept including product details and a pitch outline.

## 🚀 How to Run the Project
1 Install dependencies
pip install streamlit groq
2 Add your Groq API key

Replace this line in the code:

client = Groq(api_key="YOUR_API_KEY")
3 Run the app
streamlit run app.py
